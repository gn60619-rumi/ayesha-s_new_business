## ✏️ How to Edit

### Edit on GitHub (easiest)
1. Click `index.html` in this repo
2. Click the **pencil icon** ✏️
3. Make your changes
4. Click **"Commit changes"**

### Edit Locally
1. Download `index.html`
2. Open in **VS Code** or any text editor
3. Use **Ctrl+F** to find and replace content
4. Save and re-upload to GitHub

### Common Edits — Search (Ctrl+F) for:

| What to change | Search for |
|---|---|
| Phone number | `90360 33103` |
| Email address | `info@dipsons.com` |
| Office address | `Mangalwar Peth` |
| Gold/brand color | `#c8922a` |
| Navy/dark color | `#0b1e3d` |
| Page title | `<title>` |

## 🌐 Deploy on GitHub Pages (Free Hosting)

1. Go to repo **Settings**
2. Click **Pages** in the left sidebar
3. Under **Source** → select **main** branch → click **Save**
4. Wait 1–2 minutes
5. Live at: `https://yourusername.github.io/dipsons-website`
